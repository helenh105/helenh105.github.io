# Writeup and Evaluation
---

With the dataset that I was using, I was considering using the Leaflet maps that Observable frameworks had. The dataset that I utilized made it easy to access longitude & latitude points by plotting each zipcode as a circle on the map. The color encoding separates systems that are at or above the EPA lead action level (which is 0.015mg/L). The size is encoded to the amount of lead detected, so larger circles are higher lead values and smaller circles are smaller lead levels. This is useful as it differentiates to what zip codes or areas needed more attention in terms of taking action. 

Additionally, the size encoding can be a bit distracting to the viewer when they initially open up the map and may be harder to click the smaller circles to get their information as they require precise movement.

The external source I used was a dataset from a GitHub titled “Awesome Public Datasets” which were friendly with this project and open source so they were easy to access. The actual dataset used to develop this project was a portion of ZipCheckup’s data focusing on water quality across the United States through the representation of zipcodes.
